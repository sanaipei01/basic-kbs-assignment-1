## Project Description
This is a simple Knowledge-Based System that uses facts and rules to infer new information.

## Logic Implemented
- **Fact:** It is cold.
- **Rule:** IF "It is cold", THEN "People wear jackets".
- **Result:** The system correctly infers that people should wear jackets.
